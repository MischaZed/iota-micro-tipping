IOTA Micro Tips

As the first crypto currency ever, IOTA allows to send transactions of arbitrary size with zero fees. An ideal case for micro tips. This JavaScript library enables everyone with to deploy micro tipping functionality on their web application with lowest possible effort.

By simply applying data attributes (tip address, default tip amount and an optional message) to a DOM element, this library automatically integrates a control to allow micro tipping. The appearence of the micro tipping control and QR code can be fully overriden and customized. By configuring the library with an optional IOTA full node to connect to, it also allows to query and display tips already spent.
